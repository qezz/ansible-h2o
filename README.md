ansible-h2o
=========

An Ansible role of [h2o](https://h2o.examp1e.net).

Role Variables
--------------

| Variable | Description |
| -------- | ----------- |
| `h2o_version` | Version number of h2o |

Example Playbook
----------------

```yaml
    - hosts: servers
      roles:
         - { role: takebayashi.h2o }
```
